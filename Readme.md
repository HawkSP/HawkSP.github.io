# Elliott John Mitchell - Personal Portfolio

Welcome to the personal portfolio of Elliott John Mitchell, an AI & Computer Science Innovator. This fully responsive personal portfolio website is built using HTML, CSS, and JavaScript, showcasing Elliott's projects, skills, and professional background.


## Installation

To install this personal portfolio, follow these steps:

### Linux and macOS:

```bash
sudo git clone https://github.com/HawkSP/hawksp.github.io.git
```

### Windows:

```bash
git clone https://github.com/HawkSP/hawksp.github.io.git
```

### Usage

Open the `index.html` file in your browser to view the portfolio.
